162  /**
163  * The project has an array of "known" codecs, i.e. an array of
164  * created instances of codecs.
165  *
166  * By using this function, we set particular instance of codec
167  * for its index (determined by `#BTCore::Codec::getCodecIdx()`),
168  * and when `setCurrentCodecIdx()` will be called next time,
169  * exactly this instance will be used.
170  *
171  * If codec with the same index already exists as a "known" one,
172  * then after calling this method old instance is removed.
173  *
174  * This method is primarily needed to make the project behave more
175  * friendly in the following situation:
176  *
177  * - User has set up its codec as he/she wants
178  * - User selected different codec (i.e. with different index, see
179  * `#BTCore::CodecIdx`), probably accidentally
180  * - User switches back to previously used codec
181  *
182  * Without the collection of "known" codecs, after the sequence above,
183  * settings of the first codec are lost, which is frustrating.
184  *
185  * But with known codecs, settings will be kept.
186  *
187  * This known codecs collection is kept only in runtime, it isn't
188  * stored to the XML file.
189  */
190  void addKnownCodec(std::shared_ptr<Codec> p_codec);
