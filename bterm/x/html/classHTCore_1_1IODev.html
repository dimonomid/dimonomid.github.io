Detailed Description

Abstract class that represents IO device: arbitrary data can be read from it and written to it.

A pointer to an instance of concrete implementation of this abstract class should be given to HTCore::Project via HTCore::Project::setIODev(), and all communication with remote device will be done through that IO device.
