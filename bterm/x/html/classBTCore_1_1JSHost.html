Detailed Description

Class that represents JavaScript host environment for BTCore.

It provides running scripts with host object, like io, factory, etc.

see JavaScript host API

Call QJSEngine::evaluate() on host-specific QJSEngine instance.

It differs from regular QJSEngine::evaluate() as follows:

  • JSHost specifies its own host properties, see JavaScript host API
  • JSHost executes all scripts in strict mode ( https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Strict_mode )

QJSValue JSHost::callFuncWithCommonContext ( QJSValue  func,
QJSValueList  arguments,
QString  descr 
)

Calls function func with given arguments, and with common scripts context, which is given to called function as this.

Additionally it takes additional argument descr, which is used later in events. For example, if script calls io methods, which initiate message sending, then event will contain the descr string, which makes log more useful: we clearly see which message is generated by which script.
